📖 How to read this

  • 🟢 PROMOTE — sandbox ad paused + renamed PROMOTED-{name} (the sandbox version stops spending). A successful PROMOTE should also produce a CLONED→BAU entry; if you see PROMOTE without a matching CLONED→BAU, the clone step skipped (likely no bau_target_map.json entry for that cat, or a page-permission issue).
  • 🔵 CLONED→BAU — new ad created in the cat's BAU Core CBO adset using creative_id reuse, post ID preserved. Created PAUSED for human review before live.
  • 🔴 PAUSE — sandbox ad paused + renamed PAUSED-{name} (bad-performance kill).
  • 🟠 PAUSE (bulk) — ad paused without rename (e.g. mid-week 22 Jun KILL list or 23 Jun bulk-pause of low-priority cats). No PROMOTED-/PAUSED- prefix added.
  • Metrics shown are the latest sandbox-review window the ad was tracked in; "—" means no metric was captured (ad was actioned outside the review windows).
